Open on hover

This example shows the onHover option; in this case we also used a custom hoverDelay duration (the time interval before a menu opens on hover). 

$('#my-menu').tendina({
  onHover: true,
  hoverDelay: 500
});

Active menu example

In this case we're using the activeMenu option to set an open menu when Tendna will be inited. In a real world use case, we could pass the current page selector to the option in order to get the menu to show the page we're visiting.

You can use plain CSS selectors and jQuery objects as well.

<ul>
  <li>
    <a class="opeth" href="/opeth">Opeth</a>
  </li>
</ul>
$('#my-menu').tendina({
  activeMenu: $('.opeth')
  // or
  activeMenu: '.opeth'
});
