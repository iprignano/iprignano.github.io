Tendina is a simple jQuery plugin that helps you to build dynamic,
interactive side-menus in seconds.

Download

Usage

The basic Tendina use case setup consists in 3 simple steps:

1. Include jQuery and Tendina in your page (heh)

<!-- Shake your </body> -->
<script src="path/to/jquery-1.11.1.min.js"></script>
<script src="path/to/tendina.min.js"></script>

2. Set up your menu markup

The markup must be formatted as an unordered list with links, like this:

<ul id="my-menu">
  <li><a href="#">Rock</a>
    <ul>
      <li><a href="#">Garage Rock</a>
        <ul>
          <li><a href="#">The Hives</a></li>
        </ul>
      </li>
      <li><a href="#">Progressive Rock</a>
        <ul>
          <li><a href="#">King Crimson</a></li>
        </ul>
      </li>
    </ul>
  </li>
</ul>

Don't forget anchor tags!
Tendina uses them for DOM traversing, so don't forget to add them (even if they don't have any href).

3. Summon Tendina

// Wuzzah!
$('#my-menu').tendina()

You're done! Let's chill out for a sec now, it's been hard.

Cool things

  • It's lightweight (~4kb minified)
  • It's very easy to use, still has some neat customization options and features
  • Supports 3-level-deep nesting
  • It can be used to convert a horizontal multi-level menu in a mobile dropdown menu in few lines of code
  • Handles dinamically added elements (you can build menus from dynamic JSON trees, yay!)

Browser support

Tendina should work with most browsers, including IE8+ (it also depends from which jQuery you're using).

Open on hover

This example shows the onHover option; in this case we also used a custom hoverDelay duration (the time interval before a menu opens on hover). 

$('#my-menu').tendina({
  onHover: true,
  hoverDelay: 500
});

Active menu example

In this case we're using the activeMenu option to set an open menu when Tendina will be inited. In a real world use case, we could pass the current page selector to the option in order to get the menu to show the page we're visiting.

You can use plain CSS selectors as well as jQuery objects.

<ul>
  <li>
    <a class="opeth" href="/opeth">Opeth</a>
  </li>
</ul>
$('#my-menu').tendina({
  activeMenu: $('.opeth')
  // or
  activeMenu: '.opeth'
});
